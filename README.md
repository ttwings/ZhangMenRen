## 游戏背景
- 《炼器掌门人》是由ttwings独立开发的放置类游戏。玩家扮演一个炼器师，通过炼器来获得资源，来提升自己的实力，最终成为炼器界的掌门人。
## 游戏玩法
- 玩家通过采集、炼制法宝、丹药、完成委托、交易等来获得资源，提升自己的门派实力，最终借助门派和自身能力，飞升。
## 游戏特色
- 采集、炼制法宝、丹药、完成委托、交易等
- 门派、功法、神通、法宝、丹药、委托、交易等
## 游戏亮点
- 通过炼制方程式，可以炼制出各种法宝、丹药
- 法宝、丹药可以用来参悟神通
- 人物可以通过参悟神通来提升自己的实力
